Selecting attendance lists

Looked at 2 different patterns for users to generate attendance lists:

  • Search for an activity or location using an autocomplete
  • View all activities on a selected date

Recording attendance

Explored different ways of recording attendance:

  • Modal dialogs
  • Separate page

Additionally, looked at different call-to-action patterns for users to record attendance:

  • Dropdown button with 'attend' and 'not attend' statuses
  • A single 'record attendance' button

Whereabouts tiles

Navigation tiles on the main Whereabouts page have been updated:

  • Get unlock list
  • View daily schedules
  • Attend an activity
  • Mark refusals and sickness
  • Attendance data
  • Add bulk appointments

Some tiles link to revised 'selection' pages where users can choose the parameters for the subsequent 'list' pages.

List pages all follow a similar design, but with small differences to cater for different user needs.
