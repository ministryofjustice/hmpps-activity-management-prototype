Extra information (experimental)

Ability to add the same comment in one go for a whole set of appointments (during bulk appointment creation). Currently it is only possible to add comments individually for each appointment in the upload. This is experimental, still need to uncover the exact user needs here.

Tasklist (experimental)

With each new MVP+ feature (alerts, non-associations, irregular schedules, schedule view for appointment series), the create journey becomes longer and more complex. Tasklist is an experiemental solution which aims to break these steps into individual transactions.

  • Using tasklist to break up longer forms
